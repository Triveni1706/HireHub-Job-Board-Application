\# Job Board Application



A full-stack Job Board web application where recruiters can post jobs and candidates can search and apply for jobs.



---



\## 🚀 Features

\- User authentication (Login / Register)

\- Recruiters can post jobs

\- Candidates can view and apply for jobs

\- Resume upload

\- Secure REST APIs

\- Role-based access (Recruiter / Candidate)



---



\## 🛠 Tech Stack



\### Frontend

\- React

\- HTML

\- CSS

\- JavaScript



\### Backend

\- Node.js

\- Express.js

\- MongoDB

\- Mongoose
